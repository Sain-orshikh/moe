
# Mongolia Economic Olympiad Registration Website

This is the official registration website for the Mongolia Economic Olympiad, built with Next.js, TypeScript, Tailwind CSS, and pnpm. The site is bilingual (Mongolian/English) and designed to be national, institutional, prestigious, trustworthy, and modern.

## Getting Started

1. Install dependencies:
	```sh
	pnpm install
	```
2. Run the development server:
	```sh
	pnpm dev
	```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Bilingual: Mongolian and English with language toggle
- Modern, minimal, and institutional design
- Built with Next.js App Router, TypeScript, Tailwind CSS
- Production-ready and scalable

## Folder Structure
- `src/app` — Main application code
- `src/app/layout.tsx` — Root layout
- `src/app/page.tsx` — Home page
